Beat Maker

Acerca

Este projecto foi elaborado no âmbito da disciplina de Laboratórios de Informática do MIECT na Universidade de Aveiro.

O projecto consiste na implementação de uma aplicação web não nativa que consiga criar excertos de música, importar e exportar ficheiros audio, entre outros.

Este trabalho tem como objectivo a consolidação de conhecimentos sobre as matérias estudadas ao longo do semestre(Html,CSS,Javascript,Python,CherryPy,Bases de Dados, etc).

Este projecto foi desenvolvido pelos alunos Rui Machado(65081), João Vieira(50458), Pedro Cruzeiro(103173), Inês Felício(100084).
